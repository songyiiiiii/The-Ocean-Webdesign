
# 渤海海水水质数据集

## 数据来源
- 原始数据: 渤海海水水质30m栅格数据集(2017-2023)
- 处理时间: 2026-05-20 10:27:24

## 文件列表及说明

### 01_raw_data.csv - 原始数据
| 字段 | 说明 |
|------|------|
| source_layer | 数据来源图层（年月标识） |
| year | 年份 |
| month | 月份 |
| lon | 经度 |
| lat | 纬度 |
| offshore_km | 离岸距离（公里） |
| zone | 区域划分（近岸/过渡/远海） |
| 水质类别 | 水质等级 |
| pH值 | 酸碱度 |
| 溶解氧_mg_L | 溶解氧浓度 |
| 化学需氧量_mg_L | COD浓度 |
| 无机氮_mg_L | 无机氮浓度 |
| 活性磷酸盐_mg_L | 活性磷酸盐浓度 |
| 石油类_mg_L | 石油类污染物浓度 |

### 02_zone_statistics.csv - 区域统计
按近岸/过渡/远海三个区域统计各指标的均值、标准差、中位数、最大/最小值

### 03_yearly_statistics.csv - 年际统计
按年份统计各指标的年度平均值

### 04_monthly_statistics.csv - 月度统计
按月份统计各指标的月平均值

### 05_distance_gradient.csv - 梯度统计
按离岸距离区间统计各指标平均值

### 06_quality_class_statistics.csv - 水质类别统计
各区域不同水质类别的数量及占比

## 区域划分标准
- 近岸: 离岸距离 ≤ 20 km
- 过渡: 20 km < 离岸距离 ≤ 50 km  
- 远海: 离岸距离 > 50 km

## 注意事项
1. 部分监测点可能有缺失值，已保留原始状态
2. 离岸距离基于简化的海岸线控制点计算，仅供参考
3. 水质类别标准依据中国《海水水质标准》(GB 3097-1997)
